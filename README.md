<h3>Java Spring</h3>

<p>Auto Configuration -> Spring Boot looks into pom.xml and sees the pring-boot-starter-web dependency. It will automatically start an embedded Tomcat server on port 8080 and set up tools to convert Java objects into JSON.</p>
<p>@SpringBootApplication -> Scans the code for components</p>
<p>@Autowired -> Find an object from the code of a particular type and plug it here</p>
<p>@Service -> Creates bean in common memory called ApplicationContext or Inversion of Control (IoC) container.
